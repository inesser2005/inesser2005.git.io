F1TRACK

Aplicação Web Avançada para entusiastas de Fórmula 1.

Desenvolvida para a UC de Tecnologias Avançadas para Client-side, esta plataforma foca-se na visualização de telemetria e dados estatísticos da época 2025-26.

Uso de Firebase: A utilização desta framework permite aos utilizadores efetuar login de forma rápida e eficaz.

Decisão de Design: O formato exclusivo para desktop foi escolhido para permitir a visualização detalhada de gráficos de desempenho e tabelas de tempos complexos que requerem maior espaço de ecrã.

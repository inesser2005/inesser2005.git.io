F1TRACK

Este projeto criado para a UC de Tecnologias Avançadas para Client-side, para aplicar os conhecimentos obtidos ao longo do semestre.

O design desta web-aplicação foi pensada para um formato exclusivamente desktop, para garantir uma melhor experiência com a mesma. E devido á natureza de necessitar de espaço para demosntrar as suas funcionalidades e as telemetrias da própia F1. A mesma possui apenas os dados do ano 2025-26.

Sound4all

Criação de marca e plataforma de e-commerce fictícia para áudio.

Este projeto nasceu da vontade de explorar o meu estilo pessoal na criação de interfaces comerciais. O desafio foi criar uma identidade visual do zero e aplicá-la num website totalmente responsivo.

Foco Multidevice: A aplicação foi pensada para garantir uma navegação fluida em ecrãs mobile e desktop, utilizando componentes personalizados do Bootstrap para manter a consistência da marca.
